# Contributing

Thank you for your interest in this project!

## This is a Personal Lab / Portfolio

This repository documents my **DevOps Full-Cycle Homelab Platform** — a personal learning project. It is published primarily as a portfolio and knowledge base.

## How You Can Contribute

### 🐛 Found an Error or Outdated Info?

Open a GitHub Issue with:
- What is wrong / outdated
- Suggested correction

### 💡 Suggestions

Feel free to open an Issue with suggestions for:
- Additional tools/components to add
- Improvements to runbooks or documentation
- Architecture improvements

### 🔀 Pull Requests

PRs are welcome for:
- Fixing typos or factual errors in documentation
- Improving English grammar/clarity
- Adding useful commands or troubleshooting tips

**Please do NOT submit PRs that:**
- Add real credentials or secrets
- Change the core infrastructure design
- Add unrelated tools

## Setting Up the Lab Yourself

See the [Quick Start](README.md#-quick-start) section in README.md.

Minimum requirements:
- 2x bare-metal or VM hosts (16GB RAM each)
- Proxmox VE 8.x
- Basic Linux/Kubernetes knowledge

## Code of Conduct

Be respectful. This is a learning community — all skill levels are welcome.
